# install

One-line install scripts by itgoyo.

## Claude Code

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/itgoyo/install/main/cc.ps1 | iex
```

### macOS / Linux (Bash)

```bash
curl -fsSL https://raw.githubusercontent.com/itgoyo/install/main/cc.sh | bash
```
